# ThermoChain

ThermoChain é um projeto voltado para o monitoramento térmico e a rastreabilidade da cadeia de frio de vacinas.

O objetivo do sistema é garantir que vacinas sejam armazenadas e transportadas dentro da faixa de temperatura adequada, reduzindo perdas e aumentando a segurança na distribuição.

## Objetivo

Desenvolver uma solução que permita:

- Monitoramento contínuo de temperatura
- Registro digital de dados térmicos
- Rastreabilidade da cadeia de frio
- Redução de perdas de vacinas
